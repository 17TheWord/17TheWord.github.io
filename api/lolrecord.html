# 随机语音

请求方法：GET

# 随机语音JSON

请求方法：GET

文件格式：OGG

https://api.scareye.com/lolrecord?type=json

返回值：

# 提取 pic 的值
{"pic": "https://api.scareye.com/lolrecords/Ahri/File0402.ogg"}

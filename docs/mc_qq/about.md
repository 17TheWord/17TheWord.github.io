## NoneBot 端

- 插件版需要 `websockets`
- MCRcon版需要 `mcrcon`

## Minecraft Server 端

- 版本支持：
  - 理论上支持所有版本，但需要编译
  - 插件使用的接口有：
    - `AsyncPlayerChatEvent` 异步玩家聊天事件
    - `PlayerJoinEvent` 玩家加入事件
    - `PlayerQuitEvent` 玩家离开事件

# 启动器下载区

启动器资源

官方正版启动器的安装包在 资源收录区open in new window 有提供，或直接去 MC 官网open in new window 下载

国内更建议玩家使用第三方启动器，原因：

  1. 官启下载慢，需要搭配加速器食用；

  2. 官启默认将游戏下在C盘，且难以更改；

  3. 官启安装所有 第三方资源 都只能手动，非常繁琐；

  4. 官启与第三方 API（Forge、Fabric、Optifine等）兼容性不强，优化也相比 第三方启动器 稍差一些。

启动器推荐

  • 第三方启动器首推 PCL2， 资源收录区open in new window 提供了 PCL2

  • 正式版供大家下载，也可以自行前往作者的正式版帖子下载：点此进入正式版 PCL2 下载页面open in new window。

  • 也可以用 HMCL 下载， 资源收录区open in new window 里有提供，或在 HMCL 官网下载：点此进入 HMCL 官网open in new window。

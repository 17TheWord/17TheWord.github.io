# 前言

# 介绍

  • 基于 NoneBot 的与 Minecraft Server 互通消息插件

  • 支持QQ群、QQ频道

  • 项目地址：17TheWord/nonebot-plugin-mcqqopen in new window

# 功能

  • 推送消息列表

    • 服务器 -> QQ
      • [x] 加入 / 离开 服务器消息
      • [x] 玩家聊天信息
      • [x] 玩家死亡信息（死亡信息为英文，非插件服务端不适用。）
    • QQ -> 服务器
      • [x] 指令
      • [x] 群员聊天文本
      • [x] 图片、视频等内容转换为 [图片]、[视频]

  • 特殊消息支持

    • 群聊
      • [x] @ 消息
      • [x] 回复消息（转换成@消息）
    • 频道
      • [x] @ 消息
      • [x] 回复消息（转换成@消息）
    • 未支持的消息已被替换，如： [图片]、 [视频] 等等

# 介绍

基于 NoneBot 的与 Minecraft Server 互通消息插件

支持QQ群、QQ频道

推送消息列表：

  • 服务器 -> QQ：
    • 加入 / 离开 服务器消息
    • 玩家聊天信息
  • QQ -> 服务器：
    • 群员聊天文本
    • 图片、视频等内容转换为[图片]、[视频]
